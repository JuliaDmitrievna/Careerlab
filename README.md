# CAREERLAB — образовательный лендинг
Одностраничный сайт (landing page) онлайн-платформы обучения современным профессиям.  
Проект выполнен в учебных целях.

## Используемые технологии
- **HTML5** — семантическая разметка
- **CSS (Tailwind CSS)** — основной CSS-фреймворк
- **JavaScript (Vanilla JS)** — интерактивность и логика
- **GitHub Pages** — публикация проекта

Все страницы проекта используют **один CSS-фреймворк (Tailwind CSS)**.

## Структура сайта
Содержит следующие блоки:
- Hero-секция
- О нас
- Популярные курсы
- Призыв к консультации
- Тарифы (интерактивный барабан)
- Отзывы учеников
- FAQ (аккордеон)
- Контакты (форма обратной связи)
- Футер

## Отдельные страницы
### Страница авторизации
- Файл: `login.html`
- Содержит форму входа
- Использует тот же хедер и футер, что и главная страница

### Страница 404
- Файл: `404.html`
- Стилизована
- Содержит полезные ссылки:
  - возврат на главную
  - переход к контактам

## FAQ
Блок FAQ реализован в виде **аккордеона**:
- Открывается один вопрос за раз
- Используется JavaScript
- Стрелка меняет направление при открытии/закрытии

## Контакты
- Форма обратной связи
- Поле телефона с автоматическим форматированием
- Все поля валидируются
- После отправки показывается **модальное окно об успешной отправке**

## Адаптивность
- Полная адаптация под:
  - iPhone SE
  - iPhone / Pro / Pro Max
  - Планшеты
  - Десктоп
- Реализовано мобильное меню (burger menu)
- Отдельный файл `responsive.css` для медиазапросов
